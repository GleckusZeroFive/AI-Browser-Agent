# Поиск вакансий на HeadHunter (hh.ru)

Этот пример демонстрирует использование AI Browser Agent для автоматизации поиска работы на крупнейшем российском сайте вакансий hh.ru (HeadHunter).

## Возможности

### Основные функции

1. **Умный поиск вакансий**
   - Поиск по ключевым словам и должностям
   - Фильтрация по городам
   - Поиск удаленной работы
   - Фильтрация по уровню (Junior, Middle, Senior)

2. **Фильтрация по зарплате**
   - Минимальная зарплата
   - Диапазон зарплат
   - Валюта (рубли, доллары, евро)

3. **Анализ вакансий**
   - Чтение описаний
   - Выделение ключевых требований
   - Сравнение вакансий
   - Выявление трендов

4. **Дополнительные фильтры**
   - Опыт работы
   - Тип занятости
   - График работы
   - Отрасль компании

## Установка и настройка

### Требования

- Python 3.10+
- Установленный AI Browser Agent (см. основной [README.md](../../README.md))

### Быстрый старт

```bash
# Из корневой директории проекта
python examples/job_search/search_hh.py
```

## Рекомендации по использованию

### Для эффективного поиска

1. **Будьте конкретны в запросах**
   - Указывайте точную должность
   - Задавайте диапазон зарплаты
   - Указывайте город или формат работы

2. **Используйте фильтры**
   - "с опытом от 2 лет"
   - "только удаленка"
   - "с зарплатой от X"

3. **Анализируйте тренды**
   - Спрашивайте о популярных требованиях
   - Сравнивайте разные направления
   - Изучайте уровни зарплат

### Советы по поиску работы

1. **Изучайте требования**
   - Агент может выделить ключевые навыки
   - Составьте список необходимых технологий
   - Найдите пробелы в своих знаниях

2. **Сравнивайте предложения**
   - Не только по зарплате
   - Условия работы
   - Технологический стек

3. **Регулярность**
   - Проверяйте новые вакансии ежедневно
   - Используйте агента для быстрого скрининга

## Частые проблемы

### Много результатов поиска

**Проблема:** Агент находит слишком много вакансий.

**Решение:**
```
Вы: Слишком много, отфильтруй по зарплате от 200к и опыту 3+ года
```

### Не находит удаленные вакансии

**Проблема:** Агент показывает только офисные вакансии.

**Решение:**
```
Вы: Покажи только удаленные вакансии
# или
Вы: Найди вакансии с пометкой "remote"
```

### Нужна детальная информация

**Проблема:** Агент показывает только краткую информацию.

**Решение:**
```
Вы: Открой вакансию номер 3 и расскажи подробнее
# или
Вы: Покажи полное описание первой вакансии
```

### Агент неправильно понял запрос

**Проблема:** Результаты не соответствуют ожиданиям.

**Решение:**
- Переформулируйте запрос более точно
- Укажите конкретные критерии
- Попросите начать поиск заново

## Технические детали

### Как работает агент

1. **Открытие hh.ru**
   - Загрузка главной страницы
   - Поиск формы поиска

2. **Ввод запроса**
   - Формирование поискового запроса
   - Применение фильтров

3. **Анализ результатов**
   - Парсинг списка вакансий
   - Извлечение ключевой информации (название, компания, зарплата)

4. **Детальный просмотр**
   - Открытие отдельных вакансий
   - Анализ требований и условий

### Ограничения

- Работает только с веб-версией hh.ru
- Не отправляет отклики автоматически
- Требует интернет-соединение
- Зависит от структуры сайта hh.ru

## Приватность и безопасность

- Агент не собирает личные данные
- Не отправляет отклики без вашего согласия
- Все действия логируются локально
- Не требует авторизации на hh.ru (для просмотра)

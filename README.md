# AI Browser Agent - Додо Пицца

AI агент для автоматизации заказа еды через браузер.

## Быстрый старт

```bash
source venv/bin/activate
python main.py
```

## Структура проекта

```
ai-browser-agent/
├── main.py              # Точка входа
├── src/                 # Исходный код
│   ├── agent/           # AI агент
│   │   ├── ai_agent.py  # Взаимодействие с LLM
│   │   └── action_executor.py
│   ├── tools/           # Инструменты браузера
│   │   └── browser_tools.py
│   └── config.py        # Конфигурация
├── tests/               # Тесты
│   └── integration/     # Интеграционные тесты
├── docs/                # Документация
├── logs/                # Логи работы
├── screenshots/         # Скриншоты
└── archive/             # Устаревшие файлы
```

## Конфигурация

Файл `src/config.py`:
- **MODEL_NAME**: `llama-3.3-70b-versatile` (Groq Production)
- **GROQ_API_KEY**: API ключ Groq
- **BROWSER_HEADLESS**: `False` (видимый браузер)

## Возможности

- Диалоговый интерфейс на естественном языке
- Автоматический поиск блюд по составу
- Проверка аллергий и ограничений
- Автопродолжение при длительном поиске (до 30 действий)
- Работа с модальными окнами сайта

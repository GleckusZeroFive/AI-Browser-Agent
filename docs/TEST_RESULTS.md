## ✅ Тест 2: Обработка лимита действий

### Проверено:
- ✅ Предупреждение добавляется на 9-м действии (осталось 2)
- ✅ При достижении лимита вызывается финальный запрос к агенту
- ✅ Агент получает контекст с предупреждением

### Симуляция работы:

```
Действие 1-8:  ✓ Нормальное выполнение
Действие 9:    ⚠️ ПРЕДУПРЕЖДЕНИЕ - осталось 2 действий
Действие 10:   ⚠️ ПРЕДУПРЕЖДЕНИЕ - осталось 1 действий
Действие 11+:  ❌ ЛИМИТ - агент объясняет пользователю ситуацию
```

### Изменения в коде:

**До:**
```python
if actions_count >= max_actions:
    print("\n⚠️  Достигнут лимит действий. Возвращаюсь в диалог.\n")
    # Агент молча "падает"
```

**После:**
```python
# Предупреждение
if actions_count >= max_actions - 2:
    context += f"\n\n⚠️ ВНИМАНИЕ: Осталось {max_actions - actions_count} действий..."

# Graceful handling
if actions_count >= max_actions:
    print("\n⚠️ Достигнут лимит действий.")
    final_response = self.agent.chat(
        "Достигнут лимит действий (10). Объясни пользователю, что произошло и что удалось сделать."
    )
    print(f"\n🤖 Агент: {final_response}\n")
```

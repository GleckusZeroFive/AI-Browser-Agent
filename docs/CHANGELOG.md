# Changelog

## 2026-01-14 - Улучшения работы агента

### Добавлено
- **Новое действие `find_text`** - поиск текста на странице через Ctrl+F
  - Агент теперь использует браузерный поиск вместо несуществующей кнопки поиска
  - Метод `find_text_on_page()` в [browser_tools.py](src/tools/browser_tools.py:261)
  - Добавлено в action_map в [action_executor.py](src/agent/action_executor.py:23)

### Улучшено
- **Обработка лимита действий**
  - Агент получает предупреждение за 2 действия до лимита
  - При достижении лимита агент объясняет пользователю, что произошло
  - Изменения в [dialogue_manager.py](src/dialogue_manager.py:98) - метод `_execute_action_with_followup()`

### Исправлено
- Агент больше не пытается найти несуществующую кнопку поиска на сайте
- Агент не "падает молча" при достижении лимита действий

### Технические детали
- Лимит действий: 10 последовательных команд
- Предупреждение выдаётся на 8-м действии
- При лимите агент получает финальный запрос на объяснение ситуации

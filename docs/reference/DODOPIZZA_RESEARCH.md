## 3. Селекторы и взаимодействие

### ✅ Проверенные рабочие селекторы:

#### Клик по товару
```python
# По тексту названия (работает!)
await tools.click_by_text("2 пиццы")
await tools.click_by_text("Пепперони")
await tools.click_by_text("Маргарита")
```

#### Добавление в корзину
```python
# Кнопка "В корзину" (работает!)
await tools.click_by_text("В корзину")

# Force click при необходимости
await tools.click_by_text_force("В корзину")
```

#### Закрытие модального окна
```python
# ESC закрывает модалку
await tools.press_key("Escape")
```

### ⚠️ Проблемы парсинга

**Названия товаров:**
Метод `get_menu_items()` не корректно парсит названия (возвращает "Без названия").

**Причина:** Нужно улучшить селекторы для извлечения названий из карточек.

**Решение:**
- Использовать `click_by_text()` с известными названиями
- Или улучшить парсинг в `browser_tools.py`

## 6. Ограничения и edge cases

Многие комбо имеют диапазон цен (например, "939-1,158₽"), так как:
- Цена зависит от выбранных продуктов
- Есть разные размеры

### Модальные окна

- Могут содержать дополнительные опции выбора

# Исправление: Агент молчит при достижении лимита действий

**Дата:** 2026-01-14
**Проблема:** Агент выполняет 8-9 действий, но не выдаёт результат пользователю
**Статус:** ✅ Исправлено

---

## 🐛 Проблема

### Описание
Когда агент выполнял много действий подряд (7-9 действий) и подходил к лимиту 10 действий, он **молча останавливался** без вывода результата пользователю.

### Пример из реальной сессии:
```
👤 Пользователь: Хочу с девушкой закуски под фильм
🤖 Агент: Давай найдём!

[Агент выполняет:]
1. navigate → dodopizza.ru
2. find_text → "меню"
3. click_by_text → открыл раздел
4. scroll_down
5. scroll_down
6. scroll_down
7. scroll_down
8. find_text → "закуски"
9. click_by_text → открыл закуску

[Агент молчит 😶]

👤 Пользователь: А дальше?
🤖 Агент: [пустой ответ]

👤 Пользователь: Ты упал?
```

### Техническая причина

В [src/dialogue_manager.py](src/dialogue_manager.py) старая логика:

1. Агент выполнял действия в цикле `while current_action and actions_count < max_actions`
2. На действии 8 получал предупреждение: "Осталось 2 действия"
3. Продолжал выполнять действия 9, 10...
4. После действия 10 условие `actions_count < max_actions` становилось `False`
5. Цикл завершался **БЕЗ ВЫВОДА РЕЗУЛЬТАТА**
6. Управление возвращалось в `_dialogue_loop`, который ждал следующего ввода
7. Пользователь не получал информацию о том, что нашёл агент

**Счётчик инкрементировался ПОСЛЕ проверки следующего действия**, что приводило к тому что агент выполнял 10 действий, но не успевал выдать результат.

---

## ✅ Решение

### Изменения в логике

Переработана система предупреждений и принудительного отчёта:

#### 1. **Действие 8 (осталось 2)** - Мягкое предупреждение
```
⚠️ ВНИМАНИЕ: Осталось 2 действия до лимита.
Если нашёл информацию - ОБЯЗАТЕЛЬНО сообщи пользователю ЧТО именно нашёл.
Не молчи!
```

#### 2. **Действие 9 (осталось 1)** - Критическое предупреждение
```
🚨 КРИТИЧНО: Осталось 1 действие!
СЕЙЧАС ЖЕ сообщи пользователю результат в текстовом виде (БЕЗ JSON).
Не выполняй больше действий - только ответь пользователю!
```

#### 3. **Действие 10 (лимит достигнут)** - Принудительный отчёт
```python
# Лимит достигнут - ПРИНУДИТЕЛЬНО формируем отчёт
print("\n⚠️ Достигнут лимит действий. Агент подводит итоги...")
final_response = self.agent.chat(
    f"Лимит действий достигнут. Сообщи пользователю ЧТО ты нашёл за эти {actions_count} действий."
)
print(f"\n🤖 Агент: {final_response}\n")
break
```

### Ключевые изменения в коде

#### [src/dialogue_manager.py](src/dialogue_manager.py) - строки 165-185

**Было:**
```python
# Формируем контекст для агента
context = self._format_action_result(result, current_action)

# Предупреждение о лимите
if actions_count >= max_actions - 2:
    context += f"\n\n⚠️ ВНИМАНИЕ: Осталось {max_actions - actions_count} действий..."

# Отправляем результат агенту
response = self.agent.chat("Результат выполнения действия. Что дальше?", context=context)

# Проверяем следующее действие
next_action = self.agent.parse_action(response)

if next_action:
    current_action = next_action
    actions_count += 1  # ← Счётчик увеличивается ПОСЛЕ выполнения
else:
    print(f"\n🤖 Агент: {response}\n")
    current_action = None
```

**Стало:**
```python
# Формируем контекст для агента
context = self._format_action_result(result, current_action)

# Увеличиваем счётчик СРАЗУ после выполнения действия
actions_count += 1

# Предупреждение о лимите с разной степенью настойчивости
remaining = max_actions - actions_count
if remaining == 2:
    context += "\n\n⚠️ ВНИМАНИЕ: Осталось 2 действия. Если нашёл - сообщи пользователю!"
elif remaining == 1:
    context += "\n\n🚨 КРИТИЧНО: Осталось 1 действие! СЕЙЧАС ЖЕ сообщи результат (БЕЗ JSON)!"
elif remaining <= 0:
    # Принудительный отчёт
    print("\n⚠️ Достигнут лимит действий. Агент подводит итоги...")
    final_response = self.agent.chat(
        f"Лимит достигнут. Сообщи пользователю ЧТО ты нашёл за эти {actions_count} действий."
    )
    print(f"\n🤖 Агент: {final_response}\n")
    break

# Отправляем результат агенту
response = self.agent.chat("Результат выполнения действия. Что дальше?", context=context)

# Проверяем следующее действие
next_action = self.agent.parse_action(response)

if next_action:
    current_action = next_action
else:
    print(f"\n🤖 Агент: {response}\n")
    current_action = None
```

## 📊 Как работает новая логика

### Timeline выполнения действий:

```
Действие 1: navigate
Действие 2: get_page_text
Действие 3: click_by_text
Действие 4: scroll_down
Действие 5: scroll_down
Действие 6: find_text
Действие 7: click_by_text

Действие 8: get_modal_text
  ↓
  actions_count = 8
  remaining = 2
  ⚠️ Агент получает: "Если нашёл - сообщи пользователю!"
  ↓
  Агент решает продолжить (ещё 2 действия)

Действие 9: close_modal
  ↓
  actions_count = 9
  remaining = 1
  🚨 Агент получает: "СЕЙЧАС ЖЕ сообщи результат (БЕЗ JSON)!"
  ↓
  Агент ДОЛЖЕН ответить текстом, но может попытаться сделать ещё 1 действие

Действие 10: (если агент всё-таки попытался)
  ↓
  actions_count = 10
  remaining = 0
  🛑 ПРИНУДИТЕЛЬНЫЙ ОТЧЁТ
  ↓
  print("⚠️ Достигнут лимит. Агент подводит итоги...")
  ↓
  final_response = agent.chat("Сообщи что нашёл")
  ↓
  print(f"🤖 Агент: {final_response}")
  ↓
  break (выход из цикла)
  ↓
  Возврат в _dialogue_loop
  ↓
  👤 Пользователь видит результат!
```

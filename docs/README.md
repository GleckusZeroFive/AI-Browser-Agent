# AI Browser Agent для Додо Пиццы

Интеллектуальный агент для заказа еды через браузер с использованием AI.

## Возможности

- 🤖 **Диалоговый интерфейс** - общайтесь с AI на естественном языке
- 🍕 **Умные рекомендации** - агент знает меню и подберёт оптимальный вариант
- 🌐 **Автоматизация браузера** - агент сам управляет браузером
- 💬 **Два режима работы**:
  - Режим уточнения - задаёт вопросы для выбора
  - Режим действия - выполняет команды в браузере

## Технологии

- **Python 3.12**
- **Z.AI API** (glm-4.6v-flash) - бесплатная мультимодальная модель
- **Playwright** - автоматизация браузера
- **OpenAI-совместимый API**

## Использование

```bash
# Запуск агента
python main.py
```

## Примеры диалогов

**Пример 1:**
```
Вы: Хочу пиццу на троих
Агент: Для троих идеально подойдёт комбо "3 пиццы" за 929-1,047₽.
```

**Пример 2:**
```
Вы: Закажи что-нибудь
Агент: Сколько человек будет?
Вы: Четверо
Агент: Рекомендую "2 пиццы и напиток"...
```

## Архитектура

- **DialogueManager** - главный оркестратор
- **AIAgent** - интеллектуальный помощник (Z.AI)
- **ActionExecutor** - исполнитель команд
- **BrowserTools** - автоматизация браузера (Playwright)

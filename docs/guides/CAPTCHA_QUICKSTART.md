# Быстрый старт: Обработка капчи

## TL;DR (Коротко)

**Проблема**: Агент блокируется капчами на сайтах.

**Решение**: Обновлённый агент автоматически детектирует и обрабатывает капчи.

```bash
# 1. Обновите код (уже сделано, если читаете это)
# 2. Запустите агента как обычно
python main.py

# 3. Когда появится капча, агент покажет:
⚠️  Обнаружена Google reCAPTCHA v2
⏳ Пожалуйста, решите капчу вручную в браузере
⏱️  Ожидание: 300 секунд

# 4. Решите капчу в браузере
# 5. Агент автоматически продолжит работу
✅ Капча решена за 23.4с
```

**Вот и всё!** Больше ничего настраивать не нужно.

## Настройка (опционально)

### Изменить таймаут ожидания

В файле `.env`:
```env
# Увеличить до 10 минут
CAPTCHA_MANUAL_TIMEOUT=600
```

### Отключить автообработку

В файле `src/config.py`:
```python
CAPTCHA_AUTO_HANDLE = False
```

Или программно:
```python
tools = BrowserTools()
tools.auto_handle_captcha = False
```

### Использовать платный сервис

В файле `.env`:
```env
CAPTCHA_SOLVER_SERVICE=2captcha
CAPTCHA_SOLVER_API_KEY=your_api_key_here
```

*Примечание: Автоматическое решение требует платной подписки (~$3 за 1000 капч)*

## Часто задаваемые вопросы

### Q: Капча всё равно блокирует агента

**A**: Попробуйте:
1. Убедитесь, что `CAPTCHA_AUTO_HANDLE=true` в `.env`
2. Запустите с видимым браузером (`headless=False`)
3. Проверьте логи: `tail -f logs/errors.log`
4. Увеличьте таймаут: `CAPTCHA_MANUAL_TIMEOUT=600`

### Q: Капча не детектируется

**A**: Возможно, это новый тип капчи:
1. Сделайте скриншот: `await tools.take_screenshot()`
2. Проверьте вручную: `await tools.check_for_captcha()`
3. Создайте issue с описанием и скриншотом

### Q: Как узнать, какая капча на сайте?

**A**: Используйте скрипт проверки:
```python
tools = BrowserTools()
await tools.start_browser()
await tools.navigate("https://your-site.com")

result = await tools.check_for_captcha()
print(result)  # Покажет тип капчи
```

### Q: Можно ли обойти капчи без ручного решения?

**A**: Есть два способа:
1. **Минимизировать появление** - используйте антидетект (уже включён)
2. **Автоматическое решение** - платные сервисы (2Captcha, AntiCaptcha)

Полностью избежать капч невозможно, но можно сильно снизить их частоту.

### Q: Какова стоимость автоматического решения?

**A**: Примерные цены:
- 2Captcha: ~$2.99 за 1000 reCAPTCHA
- AntiCaptcha: ~$1.60 за 1000 капч
- hCaptcha дороже: ~$3-5 за 1000

Для большинства задач хватает ручного решения.

### Q: Безопасно ли использовать антидетект?

**A**: Да, абсолютно безопасно:
- Не нарушает законы
- Не взламывает сайты
- Просто имитирует обычного пользователя
- Используется всеми крупными скрейперами

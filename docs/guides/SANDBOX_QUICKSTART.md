# Sandbox Mode - Быстрый старт

## Что это?

Режим, где агент **сам тестирует свой функционал** и создаёт отчёт о том, что работает, а что сломано.

## Как запустить?

```bash
python main.py --sandbox
```

## Что произойдёт?

1. Браузер откроется
2. Агент протестирует все свои действия (navigate, click, scroll и т.д.)
3. Создастся отчёт в `data/sandbox_reports/`
4. В консоли покажется сводка

```

======================================================================

======================================================================

======================================================================
   🔧 Выполнение: navigate
   ✅ Успех

...

📊 ИТОГОВАЯ СВОДКА
```

## Зачем это нужно?

- ✅ Проверить что всё работает перед деплоем
- ✅ Автоматически найти баги
- ✅ Понять какие функции надо чинить
- ✅ Собрать статистику ошибок

## Где смотреть отчёты?

```bash
cd data/sandbox_reports/
ls -lt  # Последний отчёт сверху

# Посмотреть сводку
cat sandbox_report_*.json | jq .summary

# Найти проблемные функции
cat sandbox_report_*.json | jq '.functionality_map'
```

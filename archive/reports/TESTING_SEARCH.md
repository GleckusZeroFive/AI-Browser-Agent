## Тесты

### Тест 1: Автоматический тест (рекомендуется)

```bash
python test_search_dodopizza.py
```

**Что проверяет:**
- Запуск браузера
- Переход на dodopizza.ru
- Автоматический поиск через `search_and_type("пепперони")`
- Проверка переключения на браузерный поиск (Ctrl+F)
- Клик по найденному блюду
- Открытие модального окна

### Тест 2: Ручное тестирование через агента

```bash
python main.py
```

**Команды для проверки:**

#### Сайты С полем поиска:
```
Вы: Найди пиццу на Delivery Club в Москве
Вы: Найди роллы на Яндекс.Еде в Санкт-Петербурге
```

**Ожидаемое поведение:**
- Агент найдёт поле поиска
- Введёт текст в поле
- Покажет результаты

#### Сайты БЕЗ поля поиска:
```
Вы: Найди пепперони на додопицце в Москве
Вы: Найди четыре сыра на dodopizza.ru
```

**Ожидаемое поведение:**
- Агент попробует найти поле поиска → не найдёт
- Автоматически переключится на браузерный поиск (Ctrl+F)
- Сообщит: "Поле поиска не найдено, но текст 'X' найден на странице через браузерный поиск (N совпадений)"
- Использует `click_by_text()` для клика по найденному блюду

## Проверка результатов

### ✅ Успешный тест

**Агрегаторы (с полем поиска):**
```
Агент: Ввели 'пицца' в поле поиска
Статус: success
Метод: поле поиска
Селектор: input[placeholder*="Найти"] (или аналогичный)
```

**Додо Пицца (без поля поиска):**
```
Агент: Поле поиска не найдено, но текст 'пепперони' найден на странице через браузерный поиск (3 совпадений)
Статус: success
Метод: page_search
Найдено: true
Совпадений: 3
```

### ❌ Возможные проблемы

**Проблема:** "Не удалось найти поле поиска... Браузерный поиск также не сработал"
**Причина:** Текст действительно отсутствует на странице
**Решение:** Попробуйте другой поисковый запрос или прокрутите страницу

**Проблема:** "Текст не найден на странице через браузерный поиск"
**Причина:** Элементы загружаются динамически (lazy loading)
**Решение:** Прокрутите страницу вниз перед поиском

## Технические детали

### Стратегия поиска

1. **Попытка 1:** Найти поле поиска через 20+ селекторов
   - `input[placeholder*="Найти"]` (Delivery Club)
   - `input[placeholder*="Искать"]` (Яндекс.Еда)
   - `input[role="combobox"]`
   - `[role="searchbox"]`
   - И другие...

2. **Попытка 2:** Клик по кнопке поиска (если есть)
   - `button:has-text("Найти")`
   - `button:has-text("Поиск")`
   - И другие...

3. **Попытка 3 (НОВОЕ):** Браузерный поиск
   - Вызов `find_text_on_page(text)`
   - Использование Ctrl+F
   - Подсчёт совпадений

### Поддерживаемые сайты

✅ **С полем поиска:**
- Delivery Club (delivery-club.ru)
- Яндекс.Еда (eda.yandex.ru/[город]?shippingType=delivery)
- 2ГИС (2gis.ru)

✅ **Без поля поиска (браузерный поиск):**
- Додо Пицца (dodopizza.ru)
- Другие сайты ресторанов без поля поиска

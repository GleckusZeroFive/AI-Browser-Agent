# Исправление: Агент нетерпелив при презентации блюд

## Проблема

Агент ShoppingAgent был слишком нетерпеливым при работе с результатами поиска:

1. **Кликал сразу без ожидания загрузки**: После показа вариантов пользователю агент сразу пытался кликнуть на выбранное блюдо, не дожидаясь полной загрузки элементов страницы
2. **Не ждал открытия модального окна**: Пытался прочитать состав (`get_modal_text`) до того как модальное окно полностью загрузилось
3. **Не умел правильно обрабатывать ошибки**: При неудачном клике начинал поиск заново на другом сайте вместо повторной попытки с ожиданием

### Пример проблемного поведения

```
User: "Пепперони хочу"
Agent: [Показал 3 варианта]
User: "Пепперони премиум"
Agent: {"action": "click_by_text", ...}  ← Элемент ещё не загружен!
Result: ❌ Timeout
Agent: {"action": "get_modal_text", ...}  ← Модалка не открылась!
Result: ❌ Modal not found
Agent: "Не нашёл. Открываю другой сайт..."  ← Неправильная реакция!
```

## Решение

Обновлён промпт `ShoppingAgent` в файле [`src/agent/specialized_agents.py`](src/agent/specialized_agents.py) с добавлением:

### 1. Обязательная последовательность из 5 шагов

Добавлена секция **"⚠️ КРИТИЧЕСКИ ВАЖНО: ОЖИДАНИЕ ЗАГРУЗКИ ЭЛЕМЕНТОВ"** с чёткой последовательностью действий:

```
1. scroll_down - докрутить к элементу
2. wait_for_text - дождаться загрузки элемента
3. click_by_text - кликнуть по элементу
4. wait_for_modal - дождаться открытия модального окна
5. get_modal_text - прочитать состав блюда
```

### 2. Два сценария презентации

Разделены два разных сценария работы:

**СЦЕНАРИЙ 1**: Показать несколько вариантов для выбора
- НЕ открывать карточки блюд
- Просто показать список из `get_page_text`
- Дождаться выбора пользователя

**СЦЕНАРИЙ 2**: Презентовать одно блюдо с составом (после выбора)
- Использовать последовательность из 5 шагов
- Открыть карточку
- Прочитать и показать состав

### 3. Примеры правильного и неправильного поведения

Добавлены наглядные примеры:

✅ **ПРАВИЛЬНО**:
```
User: "Пепперони премиум"
Agent: scroll_down → wait_for_text → click_by_text → wait_for_modal → get_modal_text
→ Презентация с составом ✅
```

❌ **НЕПРАВИЛЬНО**:
```
User: "Пепперони премиум"
Agent: click_by_text (без ожидания!)
→ Ошибка! Элемент не загружен!
```

### 4. Обработка ошибок

Если что-то пошло не так:
- НЕ уходить на другой сайт
- Повторить попытку с ожиданием
- Использовать более короткий текст для клика

- ❌ Пытаться читать модалку без `wait_for_modal`
- ❌ Начинать поиск заново на другом сайте
- ❌ Открывать карточки при показе вариантов
